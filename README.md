# Meritt Restaurant 

A simple web application designed for seamless menu browsing and food ordering. Built with a focus on data integrity and user experience, this platform allows customers to explore detailed ingredient lists for every dish before placing their orders.

Features
* Menu Exploration: Browse a dynamic list of available food items.
* Ingredient Transparency: View a detailed breakdown of ingredients for every menu item to assist with dietary preferences or restrictions.
* Ordering System: Integrated shopping functionality allowing users to select items and place orders directly through the interface.

Development Philosophy
This project utilizes a Code-First Entity Framework methodology. The database schema is defined and maintained through C# classes and migrations, ensuring the data layer remains synchronized with the application logic without manual SQL scripting.

#### Technologies Used 

Framework: ASP.NET Core MVC
Database Management: Entity Framework Core (Code-First Approach)
Language: C#
Frontend: Razor Views, HTML5, CSS3, and JavaScript
Project Management Tools: Jira, Confluence, Lucidchart


### Setup 
To run this project locally, ensure you have the .NET SDK installed.

Update the database:
Run the following command in the Package Manager Console to create the database based on the code-first models:
```update-database```
